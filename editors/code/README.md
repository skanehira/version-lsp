# Version LSP for VS Code

This extension provides support for [version-lsp](https://github.com/skanehira/version-lsp).

## Features

- LSP support for `Cargo.toml`, `package.json`, `go.mod`, etc.

## Requirements

You need to have `version-lsp` installed.

```bash
cargo install version-lsp
```

## Extension Settings

This extension contributes the following settings:

* `versionLsp.serverPath`: Path to the `version-lsp` executable. Defaults to `version-lsp`.
